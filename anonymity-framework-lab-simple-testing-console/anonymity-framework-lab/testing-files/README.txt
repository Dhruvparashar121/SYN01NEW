Simplified Testing Submodule 1
==============================

The Testing Module now uses one simple console-style panel:

1. Six buttons for the six testing steps.
2. Left window: Kali terminal commands executed for the selected step.
3. Right window: Wireshark-style incoming packet/capture evidence for the selected step.
4. Previous Testing Submodule 2 Red/Blue packet visibility lab has been removed.
5. Live Fire SSH module is retained separately.

Steps covered:
- Step 1: Client wg2.conf test
- Step 2: VPS-1 relay test
- Step 3: VPS-2 relay / NAT test
- Step 4: Tor / Proxychains SSH test
- Step 5: WireGuard key verification test
- Step 6: End-to-end traceroute validation
