Geofencing Training Module (Clean Animated UI)
=============================================

This version focuses specifically on diagram clarity.

What was changed
----------------
1. All three main flowchart diagrams were regenerated.
2. Diagrams use larger clean spacing and more white space.
3. Animated packet flows are shown as moving dashed flow lines instead of dots over text.
4. Arrow directions are exact and placed at the ends of each flow segment.
5. Text is separated from animated paths to avoid clutter.
6. The rest of the UI remains a professional offline web-style learning interface.

Included required scenarios
---------------------------
1. Attacker -> Anonymity Framework -> Zombie/Bot .PK IP -> Target: PK (POF.PK)
2. Attacker --x--> Target: PK (POF.PK) [website geofenced, access denied]
3. Allowed access -> Target: PK (POF.PK):
   (a) All PK IPs
   (b) MOFA connectivity with VPN
   (c) PK:UNO

How to use
----------
1. Extract the ZIP.
2. Open index.html in a modern browser.
3. No installation or internet is required.

Purpose
-------
For defensive / educational understanding of geofencing logic and access-control decisions.
