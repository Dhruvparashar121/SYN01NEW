Testing Module Update
=====================

The previous simulated Wireshark testing panels have been removed.
The Testing Module now keeps only the Live Fire module:

Left side:
- Real local Kali/Linux terminal from the machine running npm start.

Right side:
- Apache Guacamole screen for the Ubuntu observation machine.
- Open Wireshark inside that Ubuntu desktop to view live packets.

Read README.txt and README_GUACAMOLE_LIVE_SCREEN.txt for setup commands.
