Open index.html in a browser to launch the unified Training Modules UI. Use the four module buttons to open the integrated interfaces.

Module sequence:
1. Anonymity Framework
2. Pivoting
3. Lateral Movement
4. Geofencing

The Pivoting module was added from the uploaded Pivot PPT reference and includes AI-generated SVG flowcharts, animated tunnel diagrams, an interactive workflow, lab command replay, defender correlation notes, and a quiz.

For the Anonymity Framework live terminal / Guacamole features, a compatible local server environment is still required as in the original project.
