LIVE FIRE LOCAL KALI TERMINAL - START STEPS

1. Install Node.js and npm on Kali if not already installed:
   sudo apt update
   sudo apt install -y nodejs npm

2. Extract this ZIP and open terminal inside the website folder:
   cd anonymity-framework-lab

3. Install required packages once:
   npm install

4. Start the website and the local Kali terminal bridge:
   npm start

5. Open the website on the same Kali machine:
   http://localhost:3000

6. To open it from another machine on the same trusted lab network, check Kali IP:
   hostname -I

   Then open:
   http://KALI-IP:3000

7. Go to Testing Module > Live Fire Module.
   The black Kali terminal connects automatically to the same Kali machine that is hosting the website.

WHAT CHANGED
============
- SSH connection panel removed.
- SSH target IP, username, password, and port fields removed.
- Live Fire step buttons removed.
- Real local Kali terminal added.
- Hints and expected commands update automatically as commands are executed in the embedded terminal.
- Ctrl+C button remains for interrupting commands.
- Reset Guide button resets only the hint checklist, not the shell.

SECURITY WARNING
================
This exposes a shell from the hosting Kali machine to the browser.
Only run it on a trusted, closed lab network.
Stop it with Ctrl+C in the terminal where npm start is running.
