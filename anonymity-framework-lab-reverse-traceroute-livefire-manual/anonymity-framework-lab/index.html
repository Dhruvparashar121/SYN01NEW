Demonstration Sequence

  1. Initial foothold: SSH into the manually entered host IP using the entered username and password.
  2. Direct route: Bring wg0 down and curl victim 10.0.40.100; Wireshark shows the entered host IP.
  3. VPS-1 route: Bring wg0 up; packets show 10.0.10.26.
  4. VPS-2 route: SSH into VPS-1, enable wg1; packets show 10.0.20.102.
  5. Tor route: SSH into Tor and start SOCKS tunnel; packets show 10.0.40.102.
